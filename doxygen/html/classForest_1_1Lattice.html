std::vector<unsigned int> Forest::Lattice::m_n_nodes

Number of nodes per axis.

This variable has dim components, being used as:

  1. n_nodes[0] = number of nodes in x-axis
  2. n_nodes[1] = number of nodes in y-axis
  3. n_nodes[2] = number of nodes in z-axis
    Note
    Not all the nodes are necessarily not empty

std::vector<std::vector<double> > Forest::Lattice::m_length

Minimum refinement for this assembly.

length of nodes (for Latt_type::grid)

This variable has dim vectors, having vector d size length[d].size() = n_nodes[d], being used as:

  1. length[0][i] = length of node i in x-axis
  2. length[1][i] = length of node i in y-axis
  3. length[2][i] = length of node i in z-axis
    Note
    Not all the nodes in the lattice are necessarily not empty
