Detailed Description

template<int dim>
class Forest::NeutronicModule< dim >

A Class related to neutron transport operators and solvers.

Here we solve the Neutron Transport equation

\begin{align*} \frac{1}{v(E)} \frac{\partial}{\partial t} \Psi(\mathbf{r},\Omega,E,t) + \Omega \cdot \nabla \Psi(\mathbf{r},\Omega,E,t) + \Sigma_{T}(\mathbf{r},E,t) \Psi(\mathbf{r},\Omega,E,t) = \nonumber \\ \int_{(4\pi)} \int_0^{\infty} \Sigma_{s}(\mathbf{r},\Omega^{\prime}\to\Omega,E^{\prime}\to E,t) \Psi(\mathbf{r},\Omega^{\prime},E^{\prime},t) d\Omega^{\prime} dE^{\prime} + \nonumber \\ \frac{\chi(E)}{4\pi}\int_{0}^{\infty} \nu(E^\prime)\Sigma_{f}(\mathbf{r},E^{\prime},t) \Phi(\mathbf{r},E^{\prime},t) dE^{\prime} , \end{align*}

or any approximation of it. Different methods can be used for the angular discretization, like for example the discrete ordinates, \( S_N \), or the Spherical Harmonics method, \( P_N \). Similarly, different methods can be used for the spatial approximation, as the Finite Element Method here.

In order to solve the equation, we need the following

  • State object
  • Problem Data
  • Geometry
  • Quadrature (angular discretization)
  • Method object (right now it is the neutron transport solver)

Once we have all the previous components, we are ready to solve the problem.
