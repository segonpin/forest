Detailed Description

Structure encoding the shape of the reactor.

This structure is used to see the shape of the reactor, i.e.,

for (unsigned int i = xy_shape.x_begin[], i < xy_shape.x_end[], ++i)

where the index is for the position along the 'y' axis of the 'x' line. Example of reactor, where 'X' means that there is an assembly, and '0' means that there is not assembly. 

0 1 2 3 4 - POSITION OVER X AXIS

0   0 0 X 0 0
1   0 X X X 0
2   X X X X X
3   0 X X X 0
4   0 0 X 0 0

x_begin[0] = 2; x_end[0] = 3;
x_begin[1] = 1; x_end[1] = 4;
x_begin[2] = 0; x_end[2] = 5;
x_begin[3] = 1; x_end[3] = 4;
x_begin[4] = 2; x_end[4] = 3;
