Detailed Description

We use this class to parse the command line arguments.

The usage is as follows:

int main (int argc, char **argv)
{
parse_args.exists (argv, argv + argc, "-f")
Forest::CmdOption parse_args;
assert(parse_args.exists (argv, argv + argc, "-f"));
std::string input_file = parse_args.get (argv, argv + argc, "-f");
return 0;
}
Note
Actually, the main point is specifying the input file, and getting the path. Information about the function can be found here: stackoverflow
Also boost can be very useful, especially when parsing parameter files.
