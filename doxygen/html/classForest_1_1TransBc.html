Detailed Description

template<int dim>
class Forest::TransBc< dim >

Transport operator boundary contitions.

Defines the transport operator bountary conditions

\begin{align*} L(\mathbf{r},\Omega,E)\Psi(\mathbf{r},\Omega,E) = \left( \Omega \cdot \nabla + \Sigma_{T}(\mathbf{r},E) \right) \Psi(\mathbf{r},\Omega,E) \end{align*}

with boundary conditions:

  • Incoming flux

    \begin{align*} \Psi(\mathbf{r},\Omega,E) = \Psi_{in}(\mathbf{r},\Omega,E) \quad \forall \mathbf{r} \in \partial \Omega \end{align*}

  • Albedo

    \begin{align*} \Psi(\mathbf{r},\Omega,E) = \alpha \Psi_{in}(\mathbf{r},\Omega,E) \quad \forall \mathbf{r} \in \partial \Omega \end{align*}

and apply the transport sweep to the source vector \( Q(\mathbf{r},E) \)

\begin{align*} D (L_0 + L_R) M \Phi = Q \end{align*}

and this is apply performing several sweeps

\begin{align*} \Phi^{n+1} = D L_0^{-1} (M Q - L_R M \Phi^{n}) \end{align*}

\begin{align*} \Phi^{n+1} = b - D L_0^{-1} L_R M \Phi^{n} \end{align*}

where

\begin{align*} b = D L_0^{-1} M Q \end{align*}

Note
Different energy groups are completely uncoupled from the point of view of the transport operator, so we can solve them separately.
