Detailed Description

template<int dim>
class Forest::QuadratureFactory< dim >

Generate the quadratures and return the object.

The quadrature factory is a class intended to host the functions used to create the quadratures and give them back to a base class object. Thus, this class actually needs no object, and because of this the member functions are declared as static.

Build the quadrature.

A quadrature is generated and the object is returned. The choice of the quadrature is obtained from name_, while the order of the quadrature is defined by order_. The quadrature is created by a derived class, and the object returned is assigned to the base class (which stores only the data and some members function to access it.
