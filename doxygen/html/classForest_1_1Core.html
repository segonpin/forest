std::vector<unsigned int> Forest::Core::m_n_nodes

Number of nodes per axis.

This variable has dim components, being used as:

  1. n_nodes[0] = number of nodes in x-axis
  2. n_nodes[1] = number of nodes in y-axis
  3. n_nodes[2] = number of nodes in z-axis
    Note
    Not all the nodes in the lattice are necessarily not empty

Number of nodes.

This variable has dim vectors, having vector d size length[d].size() = n_nodes[d], being used as:

  1. length[0][i] = length of node i in x-axis
  2. length[1][i] = length of node i in y-axis
  3. length[2][i] = length of node i in z-axis

std::vector<std::vector<std::vector<unsigned int> > > Forest::Core::m_indices

Each non-empty position has a unique index.

This variable has dim indices, being the first index for the z-component, second for the y-component, and the third one for the x-axis component:

  1. indices[k][:][:] = x-y plane at height k
  2. indices[k][j][:] = line parallel to x-axis at height k and ordinate j
    Note
    z and y-components run downwards from n_nodes[2] and n_nodes[1] until zero, respectively (only x-component increases).
