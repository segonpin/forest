Detailed Description

template<int dim>
class Forest::FissionSourceMatrixBuilt< dim >

Fission Source operator, matrix-built version.

Defines the isotropic fission operator to be applied to a flux vector

\begin{align*} Q(\mathbf{r},E,t) = \frac{\chi(E)}{4\pi}\int_{0}^{\infty}\nu(E^\prime)\Sigma_{f}(\mathbf{r},E^{\prime},t) \Phi(\mathbf{r},E^{\prime},t) dE^{\prime} \end{align*}

We prepare different functions to use the fission source operator in a matrix-free fashion, allowing to work with a single energy group

\begin{align*} Q_{g}(\mathbf{r},t) = \frac{\chi_{g}}{4\pi}\sum_{h \in G}\nu_{h}\Sigma_{f,h}(\mathbf{r},t) \Phi_{h}(\mathbf{r},t) \end{align*}

Remarks
This class builds the matrix if requested, but also allows matrix-free application of the fission source operator, both through the same call to vmult or vmult_add

std::vector<SparseMatrix<double> > Forest::FissionSourceMatrixBuilt< dim >::m_nusigf_matrix
private

The fission matrix.

nsigf_matrix is composed of a product of matrices, so in real we don't build the whole matrix but the necessary for its application, as it is the different fission matrices for the different energy groups and the different fission spectra, to recover the whole matrix in the following way

\begin{align*} \nu\Sigma_{f} = \left [ \begin{array}{ccc} \ \chi_{0}I & \ldots & \chi_{G-1}I \end{array} \right ] \cdot \left [ \begin{array}{c} \ \nu_{0}\Sigma_{f,0} \\ \vdots \\ \nu_{G-1}\Sigma_{f,G-1} \end{array} \right ] \end{align*}
