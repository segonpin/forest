Detailed Description

Class for the cross sections.

Here we have the cross sections for all the materials, as well as some functions to read and write this data to xml format, and some functions to check the data when enough cross sections are available, i.e., we check that \( \nu\Sigma_{f} = \nu*\Sigma_{f} \) and we check \( \Sigma_{t,g} = \sum_{h}(\Sigma_{s,g,h}) + \Sigma_{a,g} \)
