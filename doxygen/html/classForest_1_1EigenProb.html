Detailed Description

template<int dim>
class Forest::EigenProb< dim >

Eigenvalue problem; base class.

Base class to construct the derived eigenproblems. This class specifies the contract to be satisfied by the derived class. In general we are solving a generalized eigenvalue problem, but we want to rearrange the different parts to end up with a system of the form

\begin{align} A x = \lambda x , \end{align}

For instance, we can think about the generalized eigenvalue problem derived from the neutron transport equation

\begin{align} (DLM + S) \phi = \frac{1}{\lambda} X F^{T} \phi , \end{align}

This problem can be rearrange in different ways as a standard eigenvalue problem:

  • We can invert the right hand side (it must always have an inverse), so we then solve for the problem

\begin{align} A \phi = \lambda \phi , \quad \text{where} \quad A := (DLM + S)^{-1} X F^{T} . \end{align}

  • The problem can also be rearranged as

\begin{align} A f = \lambda f , \quad \text{where} \quad f := F^{T} \phi \quad \text{and} \quad A := F^{T} (DLM + S)^{-1} X . \end{align}

Here we also store some temporary vectors that we need when performing the different operations. This temporary vectors would be for

  • m_phi_tmp the scalar flux + boundary values
  • m_fission_density_tmp
  • m_fission_source_tmp
Note
In the particular case when the operator M is built by the product of two reduced rank matrices, as for example when we multiply the fission spectrum times the fission cross section, \( M = X F^{T} \), the eigenvalue can be recast in terms of the fission density \( f = F^{T}\phi \), to obtain the eigenproblem where the matrix \( A := F^{T} (DLM + S)^{-1} X \) would have the dimensions of one energy group. This can potentially increase the efficiency of the Krylov solver by reducing the dimensionality of the search space for the solution.

We update the value of the vectors using the solution.

If we have used an Arnoldi method for the eigenvalue problem, the intermediary vectors would be vectors of the krylov subspace. In order for the intermediary vectors to have a physical meaning we must perform one extra iteration using the solution vector. It have been implemented only in derived classes.

Access function for psi.

We do not have the angular neutron flux stored because of the large amount of memory resources that it might require. If anyway we want to access it, we will calculate it using the scalar neutron flux and the "incoming angular flux" at the boundaries, and the container vector must be provided from outside even if it will be resized inside this function. This operation requires one full transport sweep.
