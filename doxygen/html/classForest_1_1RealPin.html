template<int dim>
void Forest::RealPin< dim >::build_mesh ( Triangulation< dim, dim > &  tria)

Build a Triangulation<dim> object for this pin.

Depending on the type of pin, different generators are used. It should be possible to easily extend this definition to more type of pins.

Note
The triangulations associated with the pins will then be merged into a bigger triangulation (meaning that it represents an assembly or a core). Thus, one basic requirement of this way of building the mesh is that the pins can be merged together without having hanging nodes (i.e. not possible to use non-matching meshes).
